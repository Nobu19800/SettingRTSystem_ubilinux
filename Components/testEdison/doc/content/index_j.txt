はじめに
========

クラスについては、 `APIドキュメンテーション`_ に参照してください。

条件
====

testEdisonはOpenRTM-aist 1.0.0以上のC++版が必要です。

testEdisonは CMake_ を使います。CMake 2.8以上が必要です。

.. _CMAke:
   http://www.cmake.org

インストール
============

インストーラ
------------

Windowsのユーザはインストーラパッケージを使用してコンポーネントをインストール
することができます。これはコンポーネント及びそのすべての必要なライブラリを
インストールします。Windowsでインストールする場合、インストーラの使用を推奨してます。

- インストーラをダウンロードしてください。
- インストールを始めるためにインストーラをダブルクリックしてください。
- 指示にしたがってコンポーネントをインストールしてください。
- 環境変数の変更を適用するため、コンポーネントを使用する前にコンピューターを
  再起動する必要があるかもしれません。

testEdisonは ``testEdisonComp`` の実行をダブルクリックして実行することが
できます。あるいは、 ``testEdison`` を初期化関数の ``testEdisonInit`` を利用して、
マネージャにロードすることができます。

ソースから
----------

ソースを使う場合は以下の手順でインストールしてください。

- ソースをダウンロードして解凍してください::

    tar -xvzf testEdison-1.0.0.tar.gz

- 解凍されたフォルダに入ってください::

    cd testEdison-1.0.0

- ``build`` フォルダを作ってください::

    mkdir build

- `` build`` フォルダに入ってください::

    cd build

- CMakeを実行してください::

    cmake ../

- エラーが出無い場合、makeを実行してください::

    make

- ``make install`` でコンポーネントをインストールしてください。選択された
  インストール場所に書き込み権限があるかを確認してください::

  ``make install``

- インストールする場所はccmakeを実行して ``CMAKE_INSTALL_PREFIX`` を
  設定することで変更が可能です。

    ccmake ../

ここまでで、コンポーネントが使えるようになりました。コンフィグレーションは次のセクションを
参照してください。

testEdisonは ``testEdisonComp`` を実行（ ``${prefix}/components/bin`` に
インストールされます）することでスタンドアローンモードで実行することができます。
あるいは、 ``libtestEdison.so`` を初期化関数の ``testEdisonInit`` を利用して、
マネージャにロードすることができます。このライブラリは ``${prefix}/components/lib`` 
または ``${prefix}/components/lib64`` にインストールされます。


コンフィグレーション
====================

使えるコンフィグレーションパラメータは以下のテーブルを参照
してください。

================ ================== ================ ====
パラメータ       データ型           デフォルト値     意味
================ ================== ================ ====
outValueDigital  int                0                out_dpで出力する値を設定
outValuePWM      double             0                out_pwmで出力する値を設定する
================ ================== ================ ====

ポート
======

コンポーネントによって提供されるポートは以下のテーブルで述べられています。

=============== =========== ============================== ====
ポート名        ポート型    データ型                       意味
=============== =========== ============================== ====
in_adc          InPort      RTC::TimedDouble               ADCコンポーネント(ADC_Edison)の出力を受け取って表示
out_pwm         OutPort     RTC::TimedDouble               PWMコンポーネント(PWM_Edison)にデューティー比を入力
out_dp          OutPort     RTC::TimedBoolean              デジタルピン制御RTC(DigitalOutput_Edison)にHighかLowを入力する
=============== =========== ============================== ====

例
==

例のrtc.confファイルは ``${prefix}/components/share/testEdison/examples/conf/``
フォルダにインストールされています。
